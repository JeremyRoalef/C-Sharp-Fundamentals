<h1>Introduction</h1>
This repository stores sample projects on various C# concepts. Each project uses the console application on Visual Studio 2022. Additionally, each project corresponds to a video that demonstrates the project's concepts. You may find the videos under the Game Programming Student Resources Teams channel. Go to the "Demonstrations and Documentation" channel, under the shared section, and navigate to Coding/C#.

<h1>How To Use</h1>
download this repository as a zip folder. Find the green button called "<> Code", open it, and click "Download ZIP". This will download all C# projects to your computer, and you may review each project using Microsoft Visual Studio 2022 or a similar version. Each project was created using Visual Studio 2022, version 17.14.37203.1.
<br>
Additionally, you may copy and paste the .cs files for each project into your own editor and run them. You may have to do additional things that are specific for your editor to get the projects working. Alternatively, you may view the content for each C# script directly on GitHub. Navigate to one of the project folders, and review the files that end with the extension ".cs". These are the project's scripts that run the application.
