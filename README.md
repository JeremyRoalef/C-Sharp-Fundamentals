<h1>Introduction</h1>
This repository stores sample projects on various C# concepts. Each project uses the console application on Visual Studio 2022. Additionally, each project corresponds to a video that demonstrates the project's concepts. You may find the videos under the Game Programming Student Resources Teams channel. Go to the "Demonstrations and Documentation" channel, under the shared section, and navigate to Coding/C#.
